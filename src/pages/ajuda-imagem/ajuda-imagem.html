Ajuda

Para cadastrar uma imagem através de uma URL externa, é necessário entrar pelo celular ou pelo computador em um site que contenha a foto dos mangás, copiar a URL da imagem e colar no aplicativo.

Para fazer isso pelo celular, uma opção é pesquisar pelo mangá no Anilist, localizar a imagem do mangá, pressionar a imagem por um segundo, selecionar a opção Compartilhar imagem e em seguida Copiar link. Caso não consiga encontrar o botão Copiar link, há a opção selecionar a opção Abrir imagem em nova guia, ir para a guia aberta e copiar a URL. Clique aqui para ver o Anilist

Para fazer isso pelo computador, o procedimento é bem simples: primeiro entre na seção de busca de mangás de um site como o Anilist, clique com o botão direito sobre a imagem desejada e selecione Copiar endereço do link.

Não recomendamos copiar uma imagem diretamente do resultado de busca do Google, pois pode causar problemas.
